# 交易系统
## 项目概述
一个基于内存的交易管理系统，支持增、删、改和分页查询
## 技术栈
* SpringBoot 3.4.2
* jdk 21

## 其他依赖
lombok 自动生成getter/setter/注入logger等功能

## 快速开始
* 使用docker方式部署
```
./deploy/build.sh
./deploy/run.sh
```
